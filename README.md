# 🏥 Hospital Emergency Room Dashboard (Excel Project)

An interactive and data-driven **Hospital Emergency Room Dashboard** built using Microsoft Excel to analyze patient flow, wait times, satisfaction levels, and departmental referrals. This project focuses on transforming raw hospital data into meaningful insights through dynamic visualizations and dashboard design.

---

## 🚀 Key Features

* 📊 **KPIs Overview**

  * Total number of patients
  * Average wait time
  * Patient satisfaction score

* 📅 **Time-Based Analysis**

  * Monthly filtering (Jan–Dec)
  * Year selection (2023 / 2024)
  * Trend visualization for patient visits

* 🏥 **Admission Insights**

  * Admitted vs Not Admitted breakdown
  * Percentage distribution with visual comparison

* 👨‍⚕️ **Departmental Referrals**

  * Patient distribution across departments like:

    * General Practice
    * Cardiology
    * Gastroenterology
    * Renal

* 👥 **Demographics Analysis**

  * Gender distribution (Male vs Female)

* 📈 **Interactive Visualizations**

  * Pivot Tables & Pivot Charts
  * Slicers for dynamic filtering
  * Clean and modern dashboard UI
